About The WAPH Course

This page is part of a project for the Web Application Programming and Hacking (WAPH) course. The portfolio showcases skills in HTML, CSS, and JavaScript, with a focus on creating dynamic, interactive, and secure web content.

The interactive elements on this site, such as the live clocks, dynamic joke loader, and the random XKCD comic, are implemented using JavaScript to fetch and display real-time data from external APIs.

Course Instructor: Dr. Wenbing Zhao
Semester: Spring 2024
Institution: University of Dayton
